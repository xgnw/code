帮助

1.输入短域请加上http(s)://
2.中文域名请手动Punycode编码后再使用
3.网址最长支持1000字符
4.密语最长支持3000字符(合1000汉字)
5.手动填写短域以及密码为可选项目
6.密码限制2-20位(数字密码组合)/短域限制输入4位

Api接口

接口地址 https://8g.lol/api.php
注意 请使用post来访问Api
参数名 含义
url 需要缩短的网址(优先于密语)
message 需要缩短的密语
shorturl 自定义短域(可选)
passwd 加密密码(可选)
返回值(json) 含义
code 状态码,见下表
shorturl 生成的短网址,只有在code为200时才会返回

状态码释义

状态码 含义
200 成功
1002 您的IP或短域被封禁
1001 非法的输入
2001/2002 非法的自定义短域
2003 自定义短域已被使用
3001/3002 非法的密码
