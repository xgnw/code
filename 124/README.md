# html5LinkBoxPage
一个简约轻量的导航页面。两种显示模式用户可切换：搜索、分组。

### 支持页面搜索
搜索框中输入名称，相关的result块会显示在下方。

### GoToTop小火箭动效
下拉滚动条，会出现goToTop的小火箭图标，点击一下回到顶部。

### 响应式设计
响应式设计支持各类尺寸显示屏。

### 兼容性好
支持safiri/chrome/firefox
